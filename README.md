# Tic-Tac-Toe
**Tic Tac Toe Game 🎮**  A simple Tic Tac Toe game built using **Java and basic DSA concepts**. The game allows two players to play on a **3×3 grid**, taking turns placing X and O. It checks for **winning rows, columns, or diagonals** and declares a winner or draw. This project demonstrates **arrays, loops, and game logic in Java.**
